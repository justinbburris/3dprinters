# Summary

Here is a simple ABL sensor mount for the EZABL 18mm sensor to fit the Creality CR-X (over stock fan shroud).

A simple first design, printed and tested. 

# Print Settings

Printer Brand: Creality
Printer: CR-10S
Rafts: No
Supports: Yes
Resolution: 0.2 or below
Infill: 20 to 40 %
Filament_brand: PLA
Filament_color: any
Filament_material: 1.75mm

Notes: 
had to select a creality model and thingiverse not updated with CR-X as an option.  Printed on CR-X not CR-10S
